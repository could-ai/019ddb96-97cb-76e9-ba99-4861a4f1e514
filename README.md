# Demo de Flutter en Español

Esta es una aplicación de demostración de Flutter donde toda la interfaz y configuración se ha establecido en español.

## Características
- Página principal con contador.
- Interfaz limpia de Material Design.
- Totalmente en español.

## Flujos de usuario
1. Al abrir la aplicación, el usuario ve una pantalla con el título "Página Principal".
2. En el centro de la pantalla hay un texto en español indicando el número de veces que se ha presionado el botón.
3. El botón flotante con el icono "+" tiene un recuadro de información (tooltip) que dice "Incrementar" y aumenta el contador.

## Stack tecnológico
- Flutter
- Dart

## Instrucciones de ejecución
Para ejecutar este proyecto de manera local, asegúrate de tener el SDK de Flutter instalado, y luego ejecuta:
```bash
flutter run
```
